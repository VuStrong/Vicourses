# Vicourses Admin Web


## Running with Docker

```bash
docker-compose up -d
```

Open [http://localhost:4001](http://localhost:4001) with your browser to see the result.

## Running locally:

### 1. Install dependencies

```bash
npm install
```

### 2. Start the app

```bash
npm run dev
```

Open [http://localhost:4001](http://localhost:4001) with your browser to see the result.
